General procedure to run/configure/update the IFBL data portal
==============================================================

1. Initialize DB and load initial data (see DB_LOAD.txt)
2. Install the Rails app and its dependencies (see INSTALL_APP.txt) 
3. Check database.yml and Run the database migrations (GeoWebApi is needed)
4. Configure Geoserver (see GEOSERVER_CONFIG.txt)
5. Run the app (RUN.txt)
